# Analysis-and-Identification-of-Malicious-Mobile-Application
A study that examines mobile apps to detect malicious behavior by analyzing permissions, code structure, network activity, and hidden functionalities. It identifies threats like data theft, spyware, and unauthorized access using static and dynamic analysis techniques to ensure device security.
You can find dataset in kaggle link https://www.kaggle.com/datasets/saurabhshahane/android-permission-dataset
